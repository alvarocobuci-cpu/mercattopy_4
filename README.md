MercattoPy

MercattoPy é um sistema web desenvolvido com Python e Django para gerenciamento básico de uma loja.
A aplicação permite organizar categorias, produtos e vendas, além de apresentar um dashboard com indicadores importantes do negócio, como faturamento e produtos mais vendidos.

O projeto foi desenvolvido como prática de desenvolvimento web utilizando o framework Django.

Funcionalidades

Cadastro e gerenciamento de categorias

Cadastro e gerenciamento de produtos

Controle de estoque

Registro de vendas

Dashboard com indicadores de desempenho

Visualização de últimas vendas

Identificação de produtos com estoque baixo

Listagem de produtos mais vendidos

Python

Descrição

O sistema foi desenvolvido com o objetivo de simular um ambiente simples de administração de loja, permitindo o controle de produtos e vendas em um único painel.

O dashboard apresenta informações importantes como total de produtos cadastrados, número de vendas, faturamento e alertas de estoque baixo, facilitando o acompanhamento geral do funcionamento da loja.
